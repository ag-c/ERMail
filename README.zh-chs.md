# 汇编邮

是否对杂七杂八的系统自动发送的邮件感到厌烦？汇编邮是一款将这些系统自动发出的邮件进行聚合显示的邮箱客户端。在它的帮助下，你将能用统一的方式呈现最关键邮件信息，提高工作效率。
