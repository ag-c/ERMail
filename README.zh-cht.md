# 彙編郵箱

是否對雜七雜八的系統自動發送的郵件感到厭煩？彙編郵是一款將這些系統自動發出的郵件進行聚合顯示的郵箱客戶端。在它的幫助下，你將能用統一的方式呈現最關鍵郵件信息，提高工作效率。
